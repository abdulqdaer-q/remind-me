---
description: Start the Salah Times webapp
---

Start the Salah Times webapp development server from src/apps/salah-times using Vite.

Run the following command to start the webapp:
```bash
cd src/apps/salah-times && npm run dev
```

This will start the Vite development server. The webapp will be available at the URL shown in the terminal output (typically http://localhost:5173).
